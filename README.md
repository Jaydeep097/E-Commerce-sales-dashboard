# E-Commerce-sales-dashboard
Tools used: Power BI Desktop, Power Query, DAX.

An interactive Power BI dashboard designed to track and analyze e-commerce sales performance. This project transforms raw sales data into actionable insights regarding profitability, customer behavior, and regional trends.

Features
Interactive KPIs: Real-time tracking of Total Sales, Profit, Quantity, and Average Order Value (AOV).

Dynamic Filters: Slicers for filtering by State and Quarters/Months.

Customer Analysis: Identification of top-spending customers and high-performing product sub-categories.

Payment Insights: Breakdown of sales by payment mode (COD, UPI, Credit Card, etc.).

Technical Details
Data Source: CSV files (Orders & Details).

Data Transformation: Used Power Query for data cleaning, removing nulls, and formatting date types.

Modeling: Established relationships between tables to enable cross-filtering.

DAX: Implemented custom measures for business metrics like AOV.
